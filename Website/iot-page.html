物联网项目

欢迎来到我的物联网项目展示页面。这里展示了我在IoT领域的探索和创新，包括智能家居、环境监测、自动化系统等多个方面。通过这些项目，我希望能够展示物联网技术如何改变我们的生活和工作方式。

智能家居控制系统

基于Arduino和Raspberry Pi的智能家居控制系统，可以通过手机APP或语音控制家中的灯光、温度、窗帘等设备。

Arduino Raspberry Pi MQTT Node.js

环境监测站

室内外环境监测系统，可以实时监测温度、湿度、PM2.5、CO2浓度等数据，并通过云平台展示和分析。

ESP32 LoRaWAN ThingSpeak Python

智能植物养护系统

自动化植物养护系统，可以根据植物的生长情况和环境条件自动控制浇水、施肥和光照。

Arduino 土壤传感器 电磁阀 LED灯
